[Checksum](https://blog.lxvs.net/posts/repo-checksum)

Checksum is a batch script (used in Windows only). After deployment (can be uninstall cleanly), it will add several items for checksum checking (algorithms and behaviors are of your choices) to your right-click context menus.

<br/>

[Johnny's Git Kit](https://blog.lxvs.net/posts/repo-jg)

Johnny's Git Kit is a collection of customized Git/Bash commands (used in Windows only).

<br/>

[Cloud Save](https://blog.lxvs.net/posts/repo-cloudsave)

Cloud Save is a pair of scripts (used in Windows only) to link/unlink your game saves to your cloud storage services.

<br/>

[IPMI](https://blog.lxvs.net/posts/repo-ipmi)

IPMI is a script (used in Windows only), helping you use [ipmitool](https://github.com/ipmitool/ipmitool) more conveniently.
